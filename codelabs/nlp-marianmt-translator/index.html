Função do componente

Utilização do modelo MarianMT para tradução.

  • Neste exemplo a tradução é feito do inglês para o português, mas ela pode ser feita em qualquer uma das línguas suportadas pelo MarianMT.
  • Para adaptar para traduções em outras línguas é necessário verificacar se há o modelo pré-treinado disponível no MarianMT e adaptar o truncamento de strings do spacy para o idioma desejado
  • A métrica computada é o sacrebleu

Entrada esperada

Espera-se como entrada para o componente uma tabela com entradas textuais.
